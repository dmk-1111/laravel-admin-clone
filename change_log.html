V1.0.23 - 2022-11-07

New features:

  • now able to pass boolean to $form->readonly($boolean) / $form->disabled($boolan)

    ->number('comment_id', __('Comment id'))
    ->default((request()->comment_id ?? ''))
    ->readonly((request()->comment_id ? true : false));
  • added morphMany to show section

Bug fixes:

  • Number field: disable change for readonly / disabled fields
  • CSS: Space between cards
  • Show footer in card for relations inside Show / display part
  • Form Builder also check on full url when generating _pervious_ field
  • Batch Actions removed getElementClass
